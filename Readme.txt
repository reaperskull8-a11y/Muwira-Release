POS Pro v1.1.15

Terima kasih telah menggunakan POS Pro.

Untuk mulai:
1. Jalankan POS Pro Setup.exe
2. Install aplikasi
3. Buka POS Pro
4. Login menggunakan akun awal:
   Admin  : admin / admin123
   Kasir  : kasir1 / kasir123

Setelah login sebagai admin:
- Buka Profil untuk mengatur identitas toko, pajak, logo, dan user
- Buka Kelola Stok untuk menambah produk, satuan barang, dan pasok stok
- Buka Kasir untuk mulai transaksi
- Buka Laporan atau Ekspor Data untuk melihat hasil penjualan
- Gunakan Backup & Restore secara berkala

Catatan update:
- Lihat CHANGELOG.txt untuk fitur baru dan perbaikan bug
- Lihat VERSION.txt untuk info versi build
